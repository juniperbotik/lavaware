Для начала вам нужно скачать лаунчер кликнув сюда.

После вы должны скачать сам клиент кликнув сюда.

Для того чтобы боты обходили капчу вы должны купить капча гуру 5-10 рублей кликнув сюда.

После заходите на нужный сервер пишите .bots api (ваш ключ).

Теперь ваши боты обходят любую капчу кроме framed также посмотреть баланс .bots bal.
